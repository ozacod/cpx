# Pre-commit Hooks for forge-v2

This project uses pre-commit hooks to ensure code quality and consistency.

## Setup

1. Install pre-commit:
   ```bash
   pip install pre-commit
   ```

2. Install the hooks:
   ```bash
   pre-commit install
   ```

## What the hooks do

### General hooks
- **trailing-whitespace**: Removes trailing whitespace
- **end-of-file-fixer**: Ensures files end with a newline
- **check-yaml**: Validates YAML files
- **check-added-large-files**: Prevents large files from being committed
- **check-merge-conflict**: Detects merge conflict strings

### Go-specific hooks
- **go-test**: Runs all Go tests
- **go-mod-tidy**: Ensures go.mod is tidy
- **go-build**: Checks that the code compiles

## Running manually

To run all hooks on all files:
```bash
pre-commit run --all-files
```

To run specific hooks:
```bash
pre-commit run <hook-id>
```

## Updating hooks

To update to the latest versions:
```bash
pre-commit autoupdate
```

## Customization

The configuration is in `.pre-commit-config.yaml`. You can:
- Add/remove hooks
- Change hook versions
- Modify hook arguments
- Add file exclusions
